# GitPurr-pet-connect-frontend

## Descripción 

Este repositorio contiene la interfaz de usuario para el proyecto **PetConnect**. Es un sistema independiente que se comunica con la API de Spring Boot mediante servicios REST.

## Tecnología

Este proyecto ha sido inicializado utilizando las siguientes tecnologías:

  * **Framework:** React 18 + Vite.
  * **Entorno de ejecución:** Node.js.
  * **Gestor de paquetes:** npm.
  * **Configuración:** Plantilla mínima para React con Hot Module Replacement (HMR) y reglas de ESLint.

##Requisitos previos 

Para ejecutar este proyecto localmente, necesitas tener instalado:

  * **Node.js**: Versión 18 o superior.
  * **Git**: Para la clonación y gestión del repositorio.

## Cómo ejecutar el proyecto

Seguir estos pasos para levantar la aplicación en nuestro entorno local:
1.  **Clonar el repositorio:** 
2.  **Entrar al directorio del proyecto:**
    ```
    cd pet-connect-frontend
    ```
3.  **Instalar las dependencias:** 
    ```
    npm install
    ```
4.  **Levantar el servidor de desarrollo:** 
    ```
    npm run dev
    ```
5.  **Acceder a la aplicación:** 
    Abrir el navegador en: [http://localhost:5173](https://www.google.com/search?q=http://localhost:5173) 



## Información del Framework (Vite)

Esta plantilla proporciona una configuración base para trabajar con React en Vite. Actualmente, utiliza los siguientes complementos oficiales:

  * `@vitejs/plugin-react`: utiliza [Oxc](https://oxc-project.github.io/) para una transformación rápida.
  * `@vitejs/plugin-react-swc`: utiliza [SWC](https://swc.rs/) para una transformación rápida.

> **Nota sobre el Compilador de React**: El React Compiler no está habilitado por defecto en esta plantilla debido a su impacto en el rendimiento de desarrollo y construcción.
